# 買い物メモ (Shopping Memo)

シンプルで使いやすい、ブラウザで動作する買い物メモアプリケーションです。

## 特徴

- **未購入・購入済みの管理**: 買い物リストを簡単に追加・管理でき、スワイプ操作（またはボタンクリック）で状態を切り替えられます。
- **カテゴリ分類機能**: アイテムをカテゴリごとに整理して表示します。ドロップダウンから不要なカテゴリの削除も可能です。
- **数量の増減**: 各アイテムの数量をリスト上で直接 `+` `-` ボタンから変更できます。
- **翌朝リセット**: 「翌朝リセット」ボタンで、すべての購入済みアイテムを一括で未購入状態に戻し、繰り返し買うものを管理しやすくしています。
- **データ永続化**: データはブラウザのローカルストレージに保存されるため、ページをリロードしても状態が維持されます。データの一括消去機能も備えています。

## 技術スタック

- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)

## 使い方

1. `index.html` をブラウザで開きます。
2. フォームに「買うもの」「カテゴリ」「個数」を入力し、「追加」ボタンを押します。
3. リスト項目の右側にあるチェックボタン（PC）またはスワイプ操作（スマホ）で、購入済み／未購入を切り替えます。
4. 不要になった項目はゴミ箱ボタンから削除します。
5. 「翌朝リセット」ボタンを押すと、すべての購入済みアイテムが未購入リストに戻ります（日々の買い物に便利です）。
